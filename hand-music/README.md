# Hand Music

A CV/hardware project for Qualcomm's hardware track using an Arduino UNO Q with Edge Impulse

See the top-level repository README for more info

## Setup
### Hardware setup
Connect the Arduino UNO Q and webacm with the USB hub, then connect that to the laptop for power:
```
           Webcam
              |
UNO Q ---- [ USB HUB ] ---- Laptop
```
Connect to the UNO Q over the network.

### Software setup
Install the Arduino App Lab

To run this project through the App Lab:
- Zip this `hand-music` folder
- Import it to App Lab
- Press Run
